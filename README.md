\# SkinSure – Your Personal AI Skin Care Assistant



SkinSure is an AI-powered web application that analyzes skin images and predicts possible skin conditions. It helps users get quick insights about their skin using deep learning technology.



---



\## Features

\- Upload a skin image

\- AI-based skin condition prediction

\- Clean and simple user interface

\- Fast results using a trained deep learning model



---



\## Tech Stack

\*\*Frontend:\*\*

\- HTML

\- CSS



\*\*Backend:\*\*

\- Python

\- Flask



\*\*AI/ML:\*\*

\- TensorFlow / Keras

\- CNN (Convolutional Neural Network)



---



\## How It Works

1\. User uploads a skin image.

2\. The image is processed and resized.

3\. The trained CNN model analyzes the image.

4\. The system predicts the skin condition.

5\. Result is displayed on the screen.

\## Installation (Local Setup)



1\. Clone the repository:

git clone https://github.com/YOUR\_USERNAME/Skin-Sure.git





2\. Go into the project folder:

cd Skin-Sure





3\. Install dependencies:

pip install -r requirements.txt





4\. Add the trained model file:

skin\_model.h5



Place it in the root project folder.



5\. Run the app:

python app.py





6\. Open browser:

http://127.0.0.1:5000
